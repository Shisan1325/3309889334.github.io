1.杨辉三角

难度：容易

杨辉三角，又称帕斯卡三角，是一个由数字构成的三角形，其中的数字具有以下规律：每个数字都是它上方两个数字的和。

函数签名：

def generate_pascals_triangle(rows: int) -> list:
    pass

输入：

  • rows：一个正整数，表示生成杨辉三角的行数。

输出：

  • 返回一个生成的杨辉三角。

要求：

  • 如果输入的参数不是正整数，函数应抛出异常。

示例代码：

def generate_pascals_triangle(rows: int) -> list:
    # 检查输入参数是否为正整数
    if not isinstance(rows, int) or rows <= 0:
        raise ValueError("参数必须为正整数")
    
    triangle = []
    for i in range(rows):
        row = [1 if j == 0 or j == i else row[j - 1] + row[j] for j in range(i + 1)]
        triangle.append(row)
    
    return triangle

def print_pascals_triangle(triangle):
    max_width = len(" ".join(map(str, triangle[-1])))
    for row in triangle:
        row_str = " ".join(map(str, row))
        print(row_str.center(max_width))


# 测试用例
print_pascals_triangle(generate_pascals_triangle(5))
'''输出

    1    
   1 1   
  1 2 1  
 1 3 3 1 
1 4 6 4 1

'''

说明：

  • 在generate_pascals_triangle函数中，首先检查输入参数rows是否为正整数，如果不是则抛出异常。
  • 然后使用循环来逐行生成杨辉三角，每行的数字是根据规律计算得到的，其中第一个和最后一个数字都为1，其余数字为上一行对应位置和前一个位置的数字之和。
  • 返回一个生成的杨辉三角。

2.Fizz Buzz游戏

难度：容易

Fizz Buzz是一个简单的数字游戏，游戏规则如下：从1开始，逐个数数，当遇到3的倍数时，报"Fizz"，当遇到5的倍数时，报"Buzz"，当遇到即是3的倍数又是5的倍数时，报"Fizz Buzz"，其他情况报当前数字。

函数签名：

def fizz_buzz_game(n: int) -> list:
    pass

输入：

  • n：一个正整数，表示游戏进行的次数。

输出：

  • 返回一个包含n个字符串的列表，表示游戏中每次数数得到的结果。

要求：

  • 如果输入的参数不是正整数，函数应抛出异常。

示例代码：

def fizz_buzz_game(n: int) -> list:
    # 检查输入参数是否为正整数
    if not isinstance(n, int) or n <= 0:
        raise ValueError("参数必须为正整数")
    
    result = []
    for num in range(1, n + 1):
        if num % 3 == 0 and num % 5 == 0:
            result.append("Fizz Buzz")
        elif num % 3 == 0:
            result.append("Fizz")
        elif num % 5 == 0:
            result.append("Buzz")
        else:
            result.append(str(num))
    
    return result


# 测试示例
print(fizz_buzz_game(15))
# 输出：['1', '2', 'Fizz', '4', 'Buzz', 'Fizz', '7', '8', 'Fizz', 'Buzz', '11', 'Fizz', '13', '14', 'Fizz Buzz']

说明：

  • 在fizz_buzz_game函数中，首先检查输入参数n是否为正整数，如果不是则抛出异常。
  • 然后使用循环从1到n进行遍历，根据游戏规则判断每个数字的情况，并将相应的结果添加到结果列表中。
  • 返回包含游戏结果的字符串列表。

3.分发饼干游戏

难度：容易

假设你是一位很棒的家长，想要给你的孩子们一些小饼干。但是，每个孩子最多只能给一块饼干。

对每个孩子 i，都有一个胃口值 g[i]，这是能让孩子们满足胃口的饼干的最小尺寸；并且每块饼干 j，都有一个尺寸 s[j] 。如果 s[j] >= g[i]，我们可以将这个饼干 j 分配给孩子 i ，这个孩子会得到满足。你的目标是尽可能满足越多数量的孩子，并输出这个最大数值。

思路：使用贪心算法来尽可能满足更多的孩子。

函数签名：

def find_max_satisfied_children(g: list, s: list) -> int:
    pass

输入：

  • g：一个整数列表，表示每个孩子的胃口值。
  • s：一个整数列表，表示每块饼干的尺寸。

输出：

  • 返回一个整数，表示能够满足的最大孩子数量。

要求：

  • 输入的两个列表长度不超过1000，列表元素为正整数。

示例代码：

def find_max_satisfied_children(g: list, s: list) -> int:
    # 将两个列表排序
    g.sort()
    s.sort()
    
    satisfied_children = 0
    i, j = 0, 0
    while i < len(g) and j < len(s):
        if s[j] >= g[i]:
            satisfied_children += 1
            i += 1
        j += 1
    
    return satisfied_children


# 测试示例
g = [1, 2, 3]
s = [1, 1]
print(find_max_satisfied_children(g, s))
# 输出：1
'''
解释：
你有三个孩子和两块小饼干，3个孩子的胃口值分别是：1,2,3。
虽然你有两块小饼干，由于他们的尺寸都是1，你只能让胃口值是1的孩子满足。
所以你应该输出1。
'''

说明：

  • 在find_max_satisfied_children函数中，首先将两个列表g和s分别排序，以便从小到大进行匹配。
  • 使用贪心算法，使用两个指针i和j分别遍历g和s，当j对应的饼干尺寸能够满足i对应孩子的胃口时，增加满足孩子的数量，并将两个指针都向后移动。
  • 返回能够满足的最大孩子数量。

4.盛最多水的容器

难度：中等

给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

**说明：**你不能倾斜容器。

思路：使用双指针法来求解可以容纳最多水的容器。

函数签名：

def max_area(height: list) -> int:
    pass

输入：

  • height：一个整数列表，表示每个垂线的高度。

输出：

  • 返回一个整数，表示容器可以储存的最大水量。

要求：

  • 输入的列表长度不超过10000，列表元素为非负整数。

示例代码：

def max_area(height: list) -> int:
    left, right = 0, len(height) - 1
    max_water = 0
    
    while left < right:
        h_left, h_right = height[left], height[right]
        width = right - left
        curr_water = min(h_left, h_right) * width
        max_water = max(max_water, curr_water)
        
        if h_left < h_right:
            left += 1
        else:
            right -= 1
    
    return max_water


# 测试示例
height = [1, 8, 6, 2, 5, 4, 8, 3, 7]
print(max_area(height))
# 输出：49

说明：

  • 在max_area函数中，使用双指针法来求解可以容纳最多水的容器。
  • 初始化左指针left为0，右指针right为height列表的最后一个元素索引。
  • 使用h_left和h_right分别表示左右指针所指垂线的高度，width表示两个指针之间的宽度。
  • 计算当前容器的水量curr_water，并与已记录的最大水量max_water比较，取较大值。
  • 如果左指针所指垂线高度小于右指针所指垂线高度，则将左指针向右移动，否则将右指针向左移动。
  • 返回容器可以储存的最大水量。

5.最长递增子序列

难度：中等

给定一个整数数组 nums ，找到其中最长严格递增子序列的长度。

子序列 是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，[3,6,2,7] 是数组 [0,3,1,6,2,2,7] 的子序列。

函数签名：

def length_of_lis(nums: List[int]) -> int:
    pass

输入：

  • nums：整数列表

输出：

  • 返回一个整数，表示最长递增子序列的长度。
from typing import List

def length_of_lis(nums: List[int]) -> int:
    n = len(nums)
    if n == 0:
        return 0
    
    dp = [1] * n
    for i in range(n):
        for j in range(i):
            if nums[i] > nums[j]:
                dp[i] = max(dp[i], dp[j] + 1)
    
    return max(dp)


# 测试示例
print(length_of_lis([10, 9, 2, 5, 3, 7, 101, 18]))
# 输出：4

print(length_of_lis([0, 1, 0, 3, 2, 3]))
# 输出：4

print(length_of_lis([7, 7, 7, 7, 7, 7, 7]))
# 输出：1

说明：

  • 在length_of_lis函数中，首先判断输入的数组nums是否为空，如果为空，则直接返回0。
  • 创建一个动态规划数组dp，长度为输入数组nums的长度，并将每个位置的初始值设为1。
  • 遍历数组nums，对于每个位置i(0 <= i < n)，再遍历位置i之前的每个位置j(0 <= j < i)。
  • 如果nums[i] > nums[j]，则说明nums[i]可以接在nums[j]后面形成一个递增子序列，此时更新dp[i]的值为dp[j] + 1，表示以nums[i]结尾的最长递增子序列的长度。
  • 最后返回dp数组中的最大值，即为最长递增子序列的长度。

6.超级丑数

超级丑数 是一个正整数，并满足其所有质因数都出现在质数数组 primes 中。

给你一个整数 n 和一个整数数组 primes ，返回第 n 个 超级丑数 。

题目数据保证第 n 个 超级丑数 在 32-bit 带符号整数范围内。

函数签名：

def nth_super_ugly_number(n: int, primes: List[int]) -> int:
    pass

输入：

  • n：整数，表示要求的超级丑数的位置，1 <= n <= 10^6
  • primes：整数列表，表示质数数组，其中的每个元素都是质数，2 <= len(primes) <= 100，2 <= primes[i] <= 10^9

输出：

  • 返回一个整数，表示第 n 个超级丑数。

要求：

  • 时间复杂度应控制在合理范围内，以保证运行效率。
  • 考虑到 n 和 primes 的规模，可以使用最小堆和指针数组来优化计算过程。

示例代码：

from typing import List

def nth_super_ugly_number(n: int, primes: List[int]) -> int:
    # 定义一个指针数组，用于记录每个质数对应的当前最小丑数的下标
    pointers = [0] * len(primes)
    
    # 初始化结果数组和最小堆
    result = [1]
    min_heap = [(prime, i) for i, prime in enumerate(primes)]
    
    # 生成超级丑数
    while len(result) < n:
        # 取出当前最小的丑数及其对应的质数下标
        num, idx = min_heap[0]
        
        # 更新指针数组中对应质数的下标
        pointers[idx] += 1
        
        # 如果当前丑数与前一个丑数相同，跳过
        if num != result[-1]:
            result.append(num)
        
        # 更新最小堆
        for i in range(len(primes)):
            while primes[i] * result[pointers[i]] <= num:
                pointers[i] += 1
            min_heap[i] = (primes[i] * result[pointers[i]], i)
        
        # 调整最小堆，保持堆有序
        heapify(min_heap)
    
    return result[-1]


# 测试示例
print(nth_super_ugly_number(12, [2, 7, 13, 19]))
# 输出：32

print(nth_super_ugly_number(6, [2, 3, 5]))
# 输出：6

print(nth_super_ugly_number(10, [2, 3, 5]))
# 输出：12

说明：

  • 在 nth_super_ugly_number 函数中，首先定义一个指针数组 pointers，用于记录每个质数对应的当前最小丑数的下标。
  • 初始化结果数组 result 和最小堆 min_heap，最小堆中的每个元素都是 (prime, i) 的形式，表示质数和其对应的下标。
  • 进入循环，直到结果数组的长度等于 n 时停止。
  • 在每次循环中，取出当前最小的丑数 num 及其对应的质数下标 idx。
  • 更新指针数组中对应质数的下标，将其加1。
  • 如果当前丑数与前一个丑数相同，跳过，只有当丑数不同时才将其添加到结果数组中。
  • 更新最小堆，对于每个质数，将其乘以对应丑数的下一个丑数，并更新该质数在最小堆中的位置。
  • 调整最小堆，保持堆有序。
  • 返回结果数组中的最后一个元素，即为第 n 个超级丑数。
